# Futura Casa Pro v3.2 — Evolução do mapa OSM

Nova evolução aplicada:
- Lotes exibidos como áreas vetoriais clicáveis sobre o OpenStreetMap.
- Etiquetas visíveis com Quadra + Lote diretamente no mapa.
- Clique no lote para abrir detalhes, rota e ações comerciais.
- Seleção do lote diretamente pelo mapa no modo Vendas.

Observação:
- Nesta versão, as áreas vetoriais são derivadas das coordenadas centrais e das dimensões cadastradas de cada lote para melhorar a navegação comercial no mapa.
